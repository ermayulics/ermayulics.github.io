E-Surat EPD

A simple website for archiving administrative documents in the subsection of directorate administration.

For internal purposes only. Incoming and Outgoing Letters are archived on the website, so that the archiving process can be digital and easy to access and search. There are no sophisticated features yet, users can only enter existing letter information into the form and upload the scan results of the letters to be archived.

I built this in 2021 when I was still a CPNS. I built it as a requirement for basic training education assignments at the Ministry of Home Affairs. Previously I didn't have any experience in building a website, but I tried and studied diligently and self-taught with the help of tutorials from YouTube, I succeeded in making it.

For the first time I learned how to create a website. Based on the tutorial that I studied, it turns out that I can build a website using a framework, and it's easier. At that time I used the CodeIgniter3 framework and phpMyadmin as the database. Through this project, I learned about Model, View, Controller. It also deepened my use of bootstrap HTML, PHP, and my database analysis skills.

Through this website, the administrative sub-section has been able to digitize the letter archivists in the Directorate and make it easier to search for incoming and outgoing letters needed for certain purposes. This website has been used for only 1 year. This is related to the budget for purchasing a domain and hosting server which has not been facilitated by the office, which previously I spent my money on hosting and I am not willing to continue with my personal costs because this is not my personal need.
