Prodeskel Village and Subdistrict Classification Report

Displays a simple design resulting from processing village and sub-district classification data on the Prodeskel website

These are some examples of the results of data processing that I carried out using MS PowerPoint. Data is taken from Village and Subdistrict Profiles which are then downloaded in MS Excel form. The downloaded data is raw data, so I need to process the data so that it can be displayed in a presentation that is easier to understand.

I work in the Subdirectorate of Village Data and Information Management Facilitation. And we are often entrusted with processing village and sub-district data in the Village and Subdistrict Profiles so that they can be presented as a data source in presenting leadership and the needs of central government and regional government. I had to learn a lot in reading and processing complex data so that it could be simplified and presented into presentations according to the required needs.

I learned a lot in creating exposure using MS Point. At first I wasn't very familiar with it because I rarely used it before, but now I can do it better. I prefer to process data using MS Excel because it is easier to understand and display it on MS Point because it is free and can be done offline.

I still often do this, because this is one of my duties and responsibilities. We also have a team that can help each other to support joint work according to the needs and directions of the leadership. Of course, this greatly affects the performance and responsibilities of our team at the Village Data and Information Management Facilitation Sub-directorate.
